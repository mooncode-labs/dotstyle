### Checkbox Component
The Checkbox component offers a sleek and interactive way to integrate checkbox functionality into your application. It combines accessibility and style, ensuring a user-friendly experience.

#### Usage and Accessibility
- The Checkbox component is ideal for forms, settings, and any interface where binary choices are required.
- The multiple checkboxes example demonstrates how to use the Checkbox component in a group, allowing users to select multiple options and return an array of selected values.
- It features an interactive checkbox with aria-labels and attributes, enhancing both usability and accessibility.

#### Props
- label : **string**
  - The text label displayed next to the checkbox. This label is interactive and can be clicked to toggle the checkbox state.
- checked : **boolean**
  - The starting state of the checkbox. If true, the component is checked. This is an optional prop, the default will be false.
- disabled : **boolean**
  - If true, the component is disabled. This is an optional prop, the default will be false.
- onChange : **function**
  - A callback function that is called when the checkbox state changes.

#### Example
```jsx
import React from 'react';
import Checkbox from './Checkbox';

function ExampleComponent() {

  return (
    <Checkbox
      id = "enable-notifications"
      label="Enable Notifications" 
      checked={true}
      disabled={false}
      onChange={() => console.log('checked')}
    />
  );
}
```
