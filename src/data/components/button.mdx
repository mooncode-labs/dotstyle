### Button Component
The `Button` component combines the functionality of a button with the appearance of a hyperlink. It is ideal for use cases where you need a clickable element that looks like a link rather than a traditional button.

#### Usage and Accessibility
- This component is designed to function as a button and is mostly styled by our btnPrimary, btnSecondary and btnOutline.
- It includes an `aria-label` for accessibility, ensuring that the button's purpose is clear to screen reader users.
- The component can be easily disabled, allowing for flexible use in dynamic UIs.
- The utility button can be a useful approach in creating a button which regularly uses svg icons, and/or needs a small text to assist users, such as a shortcut key if set them up, or to display a message like number of messages.

#### Props
- `label`: **string**
  - The text displayed on the button.
- `onClick`: **function**
  - The function executed when the button is clicked.
- `disabled`: **boolean**
  - If set to `true`, disables the button, preventing user interaction.

<button className="btnPrimary" onClick={yourOnClickFunction}>Button</button>

<Button
  label={"dotstyle"}
  onClick={() => console.log('Button clicked')}
